Brief Introduction of Label Propagation Algorithm

 KK published on 2017-07-16 included in Machine-Learning

As I said before, I’m working on a text classification project. I use doc2vec to convert text into vectors, then I use LPA to classify the vectors.

LPA is a simple, effective semi-supervised algorithm. It can use the density of unlabeled data to find a hyperplane to split the data.

Here are the main stop of the algorithm:

  1. Let (x1,y1)...(xl,yl) (x_1,y1)...(x_l,y_l)(x1​,y1)...(xl​,yl​) be labeled data, YL=y1...ylY_L = \\{y_1...y_l\\} YL​=y1​...yl​ are the class labels. Let (xl+1,yl+u)(x_{l+1},y_{l+u})(xl+1​,yl+u​) be unlabeled data where YU={yl+1...yl+u}Y_U = \{y_{l+1}...y_{l+u}\}YU​={yl+1​...yl+u​} are unobserved, usually l≪ul \ll ul≪u. Let X={x1...xl+u}X=\{x_1...x_{l+u}\}X={x1​...xl+u​} where xi∈RDx_i\in R^Dxi​∈RD. The problem is to estimate YUY_UYU​ for XXX and YLY_LYL​.
  2. Calculate the similarity of the data points. The most simple metric is Euclidean distance. Use a parameter σ\sigmaσ to control the weights.
wij=exp(−dij2σ2)=exp(−∑d=1D(xid−xjd)2σ2)w_{ij}= exp(-\frac{d^2_{ij}}{\sigma^2})=exp(-\frac{\sum^D_{d=1}{(x^d_i-x^d_j})^2}{\sigma^2})wij​=exp(−σ2dij2​​)=exp(−σ2∑d=1D​(xid​−xjd​)2​)
