Brief Introduction of Label Propagation Algorithm

As I said before, I’m working on a text classification project. I use doc2vec to convert text into vectors, then I use LPA to classify the vectors.

LPA is a simple, effective semi-supervised algorithm. It can use the density of unlabeled data to find a hyperplane to split the data.

Here are the main stop of the algorithm:

  1. Let (x1,y1)...(xl,yl) (x_1,y1)...(x_l,y_l)(x1​,y1)...(xl​,yl​) be labeled data, YL=y1...ylY_L = \\{y_1...y_l\\} YL​=y1​...yl​ are the class labels. Let (x_l+1,y_l+u)(x\_{l+1},y\_{l+u})(x_l+1,y_l+u) be unlabeled data where Y_U=y_l+1...y_l+uY\_U = \\{y\_{l+1}...y\_{l+u}\\}Y_U=y_l+1...y_l+u are unobserved, usually l≪ul \ll ul≪u. Let X=x_1...x_l+uX=\\{x\_1...x\_{l+u}\\}X=x_1...x_l+u where x_i∈RDx\_i\in R^Dx_i∈RD. The problem is to estimate Y_UY\_UY_U for XXX and Y_LY\_LY_L.
  2. Calculate the similarity of the data points. The most simple metric is Euclidean distance. Use a parameter σ\sigmaσ to control the weights.
w_ij=exp(−d2_ijσ2)=exp(−∑D_d=1(xd_i−xd_j)2σ2)w\_{ij}= exp(-\frac{d^2\_{ij}}{\sigma^2})=exp(-\frac{\sum^D\_{d=1}{(x^d\_i-x^d\_j})^2}{\sigma^2})w_ij=exp(−σ2d2_ij​)=exp(−σ2∑D_d=1(xd_i−xd_j)2​)

Larger weight allow labels to travel through easier.

  1. Define a (l+u)\*(l+u)(l+u)\*(l+u)(l+u)\*(l+u) probabilistic transition matrix TTT
T_ij=P(j→i)=w_ij∑l+u_k=1w_kjT\_{ij}=P(j \rightarrow i)=\frac{w\_{ij}}{\sum^{l+u}\_{k=1}w\_{kj}}T_ij=P(j→i)=∑l+u_k=1w_kjw_ij​

T_ijT\_{ij}T_ij is the probability to jump from node jjj to iii. If there are CCC classes, we can define a (l+u)\*C(l+u)\*C(l+u)\*C label matrix YYY, to represent the probability of a label belong to class ccc. The initialization of unlabeled data points is not important.

  1. Propagate Y←TYY \leftarrow TYY←TY
  2. Row-normalize Y.
  3. Reset labeled data’s Y. Repeat 3 until Y converges.

In short, let the nearest label has larger weight, then calculate each label’s new label, reset labeled data’s label, repeat.
